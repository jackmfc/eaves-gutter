# Eaves Gutter & Downpipe Sizing — AS/NZS 3500.3

Standalone tool: roof catchment + rainfall → design flow (Rational Method), number of downpipes required, and an eaves-gutter capacity check. Eaves gutters to 5% AEP (1-in-20) 5-min rainfall, practical minimum design flow for blockage. Downpipe/gutter capacities are editable type defaults — confirm against AS/NZS 3500.3:2021 (Table 5.2) and the profile rating. Branded report export. Calculation aid, not design certification.
